🎨 PDF标注组件 - Web Component

支持 PDF 和 JSON 数据的传入传出，可在任何页面中复用

📖 API 使用文档

1. 基本使用

<!-- 引入依赖和组件 --> <script src="https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js"></script> <script src="pdf-annotator-component.js"></script> <!-- 使用组件 --> <pdf-annotator id="myAnnotator" style="height: 800px"></pdf-annotator>

2. 加载 PDF

// 方式1: 从URL加载 annotator.loadPDFFromUrl('document.pdf'); // 方式2: 从File对象加载 annotator.loadPDFFromFile(fileObject); // 方式3: 从Base64加载 annotator.loadPDFFromBase64(base64String);

3. 加载标注数据（JSON）

// 从JSON对象加载 const data = { rectangles: [ { id: 1234567890, page: 1, x: 100, y: 100, width: 200, height: 150, name: "重要信息", metadata: [{ key: "类型", value: "标题" }] } ] }; annotator.loadAnnotations(data);

4. 获取标注数据

const annotations = annotator.getAnnotations(); console.log(annotations); // 输出: { exportTime, totalPages, rectangles: [...] }

5. 监听事件

// PDF加载完成 annotator.addEventListener('pdf-loaded', (e) => { console.log(`PDF加载完成，共 ${e.detail.totalPages} 页`); }); // 标注变化 annotator.addEventListener('annotations-change', (e) => { console.log('标注数据:', e.detail); }); // 标注导出 annotator.addEventListener('annotations-exported', (e) => { console.log('导出成功:', e.detail); });

6. 设置 Key 预设列表

// 设置预设 key 列表，用户可以从下拉框中选择 const keys = ["姓名", "日期", "金额", "地址", "电话"]; annotator.setKeyPresets(keys); // 或者通过属性设置 <pdf-annotator key-presets='["姓名", "日期", "金额"]'></pdf-annotator>

7. 导出JSON文件

// 触发下载JSON文件 annotator.exportAnnotations();
